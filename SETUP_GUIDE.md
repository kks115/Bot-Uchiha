# SETUP GUIDE for Bot-Uchiha

## Introduction
This guide provides instructions on how to set up and run the `setup-uchiha.js` script for the Bot-Uchiha project.

## Prerequisites
Before running the setup script, ensure you have the following installed:
- Node.js (version 12 or higher)
- NPM (Node Package Manager)

## Installation
To install the necessary dependencies, run the following command in your terminal:
```bash
npm install
```

## Running the Script
To execute the `setup-uchiha.js` script, use the following command:
```bash
node setup-uchiha.js
```
Make sure you are in the directory where the script is located.
